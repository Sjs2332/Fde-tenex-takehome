# Next.js + shadcn/ui Boilerplate

A clean, premium starting point for Next.js projects featuring a modern Header, Hero, and Footer.

## Tech Stack
-   **Framework**: Next.js 16 (App Router)
-   **Styling**: Tailwind CSS 4
-   **Components**: shadcn/ui
-   **Language**: TypeScript

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Add new components**:
    ```bash
    npx shadcn@latest add [component-name]
    ```

## Project Structure
-   `src/app`: Page routes and global styles.
-   `src/components`: Custom UI components (Header, Hero, Footer).
-   `src/components/ui`: shadcn/ui primitive components.
-   `src/lib`: Utility functions.
